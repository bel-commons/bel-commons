PyBEL Web is built on PyBEL and PyBEL Tools. We have created a repository of Jupyter notebooks to demonstrate their programmatic capabilities.

Dependencies

PyBEL Web is built with a variety of tools:

Python: Flask, Requests, NetworkX, Pandas, SQLAlchemy

Web Stack: Bootstrap, D3.js, jQuery

Database: MySQL, SQLite
